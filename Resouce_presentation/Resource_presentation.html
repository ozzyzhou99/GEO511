Introduction to spatstat

spatstat is an R package used for spatial point pattern analysis. It provides a comprehensive set of tools for analyzing and visualizing spatial point patterns, performing cluster analysis, estimating intensity, and simulating point processes. The package is highly useful in a wide range of fields, such as ecology, epidemiology, and geography.

For more information, visit the official CRAN page.

Author Information

One of the primary authors of spatstat is Adrian Baddeley. He is a prominent figure in the field of spatial statistics and has contributed to several other R packages related to spatial data analysis. Adrian Baddeley is affiliated with Curtin University, Australia, and has authored multiple research papers in the domain of spatial analysis and point processes.

Features Overview

  • Visualization: Plot point patterns and various diagnostic plots to understand the spatial structure of data.
  • Spatial Analysis: Perform functions like K-function analysis and G-function to evaluate clustering or regularity.
  • Density Estimation: Estimate the density of points over a spatial region.
  • Simulation: Simulate random point patterns based on different spatial processes (e.g., Poisson process).
  • Covariate Analysis: Fit point process models using covariate data.

Data Preparation

We will load sample data and convert it into a point pattern object.

# Generate some sample data
set.seed(42)
x <- runif(100, min = 0, max = 1)
y <- runif(100, min = 0, max = 1)

# Define the window and create a point pattern object
window <- owin(c(0, 1), c(0, 1))
locations <- ppp(x, y, window = window)

# Plot the data
plot(locations, main = "Sample Point Pattern")

K-function Analysis

The K-function helps to evaluate whether the points exhibit clustering or regularity.

# Compute and plot the K-function
K <- Kest(locations)
plot(K, main = "K-function Analysis")

Density Estimation

Estimate the density of points in a given region to understand the hotspots.

# Estimate density and plot the results
density_map <- density(locations)
plot(density_map, main = "Density Estimation")

Point Pattern Simulation

Simulate a random point pattern using a homogeneous Poisson process.

# Simulate random point pattern
random_pattern <- rpoispp(100, win = window)

# Plot the simulated pattern
plot(random_pattern, main = "Simulated Point Pattern")
